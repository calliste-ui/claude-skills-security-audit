# Contributing

This repo is primarily a personal portfolio of Claude skills. That said, if you've adapted a skill or built something on top of it, feel free to open an issue to share it.

## If you find a bug or improvement

Open an issue with:
- Which skill is concerned
- What the problem or suggestion is
- If relevant, what you tested it on (Claude version, use case)

No formal process. No promises on response time.
